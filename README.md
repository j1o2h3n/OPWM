# OPWM
3D结构光的OPWM离焦投影条纹法方程的近似解

## OPWM离焦投影条纹法
OPWM离焦投影条纹法是在3D结构光投影时，投影仪在离焦情形下采用相位测量轮廓术实现三维测量的方法。其中核心思想为借鉴电力电子中最优脉宽调制（Optimal Pulse Width Modulation，OPWM）方法来消除方波的特定阶次谐波，使得离焦投影后获得高质量的正弦条纹。方法最核心的是求解方程的谐波角度问题。
